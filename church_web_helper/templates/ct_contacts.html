This page allows to download all available contacts as VCards.
It was implemented in order to ease use of ChurchTools information as phonebook.
Only name and WORK, HOME and CELL number are exported to avoid other usage.

In order to use this with automated login as CardDAV service e.g. on Fritzbox use basic Auth with URL like this
https://DOMAIN:PORT/ct/contacts?download=fritzbox
